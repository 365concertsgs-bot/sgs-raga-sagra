# SGS Raga Sagra

A healing and spiritual application built with React and Vite.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/Sgs_raga_sagra.git
cd Sgs_raga_sagra
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```
